Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language. Inside a host environment, JavaScript can be connected to the objects of its environment to provide programmatic control over them.

function greet(name) { return "Hello " + name; }
  • Client-side JavaScript
  • Server-side JavaScript
Variables

Variables are containers for storing data values.

In JavaScript, you can declare variables using:

let x = 5; const y = "Hello";
  • var - function scope
  • let - block scope
  • const - block scope, immutable
Data Types

JavaScript has 8 basic data types:

typeof "John" // Returns "string"

Primitive types:

  • String
  • Number
  • Boolean
  • Null
  • Undefined

And complex types:

  • Object
  • Array
  • Function
Functions

A function is a block of code designed to perform a particular task.

function square(number) { return number * number; }

Functions can be declared in multiple ways:

  • Function declarations
  • Function expressions
  • Arrow functions

Functions are first-class citizens in JavaScript.

Loops

Loops can execute a block of code multiple times.

for (let i = 0; i < 5; i++) { console.log(i); }

Different types of loops:

  • for - loops through a block of code a number of times
  • while - loops through a block while a condition is true

Always ensure loops have a valid exit condition to prevent infinite loops.
